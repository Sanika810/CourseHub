# CourseHub - Course Aggregation Platform

## Setup Instructions

### Backend Setup
1. Navigate to backend directory: `cd backend`
2. Install dependencies: `npm install`
3. Start MongoDB
4. Run the server: `npm run dev`

### Frontend Setup
1. Navigate to frontend directory: `cd frontend`
2. Install dependencies: `npm install`
3. Start the dev server: `npm start`

## Features
- User-submitted courses with admin approval
- Netflix-style dynamic landing page
- Advanced rating system with percentage breakdowns
- Responsive design
- Real-time notifications
- Course comparison and tracking

## Tech Stack
- Backend: Node.js, Express, MongoDB
- Frontend: React, React Router
- Authentication: JWT
